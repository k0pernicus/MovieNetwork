Presentation

Don't you have already search about a movie based by movies you really love...?

Don't you have already search about a movie based by your preferences...?


Stop it!

MovieNetwork is here to help you about these searches.

The principle is really simple: you really like a movie, and wanna see an other which look like at this one. No problem! Put the name of your favorite film in the field, and our algorithms will output all the movies which seem you like..

MovieNetwork uses the TheMovieDatabase API to work - which contains a big database of movies. We are not affiliated with them.
