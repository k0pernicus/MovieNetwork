MovieNetwork
============

What is that thing?
-------------------

**MovieNetwork** is a website which allows to display (by a **list**, or by a **graph**) similar movies, from one you like.

Programming languages
---------------------

HTML5, CSS3, Javascript, Ajax.

Libraries
---------

CSS: PureCSS

JS: JQuery, Lightbox, CoolKitten, jsPDF.

_Next: D3JS._
